# 用户手册

本手册面向博客系统的最终用户（博主），指导如何管理和使用博客。

## 快速开始

### 访问管理后台

1. 打开浏览器，访问 `https://yourdomain.com/login`
2. 输入邮箱和密码
3. 点击「登录」进入后台

### 管理后台界面

## 文章发布

### 创建新文章

1. 在左侧菜单点击「文章」
2. 点击右上角「新建文章」按钮
3. 填写文章信息

### 文章编辑器

```
┌─────────────────────────────────────────────────┐
│  标题：[                                   ]    │
│  别名：[                                   ]    │
│                                                  │
│  分类：[技术          ▼]                         │
│  标签：[+添加标签...]                          │
│                                                  │
│  ┌─────────────────────────────────────────────┐ │
│  │ # 标题                                      │ │
│  │                                             │ │
│  │ 内容支持 Markdown 语法                      │ │
│  │                                             │ │
│  │ ```php                                      │ │
│  │ echo 'Hello World';                          │ │
│  │ ```                                        │ │
│  │                                             │ │
│  └─────────────────────────────────────────────┘ │
│                                                  │
│  摘要：[                    ]                    │
│  封面：[选择图片]                               │
│                                                  │
│  状态：[草稿 ▼]  [发布] [预定时发布]          │
│  发布于：[2024-01-01 00:00]                    │
└─────────────────────────────────────────────────┘
```

### 文章字段说明

| 字段 | 说明 | 必填 |
|------|------|------|
| 标题 | 文章标题 | 是 |
| 别名 | URL 路径，默认从标题生成 | 否 |
| 分类 | 文章所属分类 | 是 |
| 标签 | 文章标签（可多选）| 否 |
| 内容 | Markdown 格式的��章内容 | 是 |
| 摘要 | 文章摘要，用于列表展示 | 否 |
| 封面 | 文章封面图 | 否 |
| 状态 | 草稿/已发布/定时发布/私有 | 是 |
| 发布时间 | 定时发布时间 | 否 |

### Markdown 语法简写

```markdown
# 一级标题
## 二级标题
### 三级标题

**粗体**
*斜体*
~~删除线~~

- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2

> 引用块

[链接文本](https://example.com)

![图片 alt](image-url)

```

代码块：

~~~markdown
```php
<?php

```
~~~

### 文章状态

| 状态 | 说明 | 可见性 |
|------|------|--------|
| 草稿 (draft) | 未发布，仅后台可见 | 仅后台 |
| 已发布 (published) | 已公开发布 | 所有人 |
| 定时发布 (scheduled) | 定时自动发布 | 发布后所有人 |
| 私有 (private) | 仅登录用户可见 | 登录用户 |

### 置顶文章

在文章列表，勾选「置顶」选项，置顶文章会始终显示在最前面。

### 批量操作

1. 勾选要操作的多个文章
2. 选择批量操作：
   - 批量发布
   - 批量删除
   - 批量移动分类

### 保存和发布

- **保存草稿**：点击「保存草稿」按钮
- **立即发布**：点击「发布」按钮
- **定时发布**：设置「发布时间」后点击「保存」

## 分类管理

### 什么是分类

分类用于组织文章，可以有层级关系。例如：

```
技术
├── PHP
│   ├── Laravel
│   └── ThinkPHP
├── 前端
│   ├── Vue.js
│   └── React
└── 数据库
    ├── MySQL
    └── PostgreSQL
```

### 创建分类

1. 点击左侧菜单「分类」
2. 点击「新建分类」
3. 填写分类信息：

```
名称：PHP
别名：php
父分类：[无 ▼]
描述：PHP 相关文章
排序：1
```

### 编辑分类

1. 在分类列表点击分类名称
2. 修改信息后点击「保存」

### 删除分类

1. 在分类列表选择要删除的分类
2. 点击「删除」按钮
3. 确认删除（文章会移动到默认分类）

## 标签管理

### 什么是标签

标签是文章的关键词，一个文章可以有多个标签。

### 创建标签

1. 点击左侧菜单「标签」
2. 点击「新建标签」
3. 填写标签信息：

```
名称：Laravel
别名：laravel（自动生成）
```

### 使用标签

在编辑文章时，从「标签」下拉框中选择，或输入新标签名。

### 标签统计

标签列表显示每个标签的文章数量，便于了解热门标签。

## 评论审核

### 评论列表

1. 点击左侧菜单「评论」
2. 查看所有评论

### 评论状态

| 状态 | 说明 |
|------|------|
| 待审核 | 新评论，等待审核 |
| 已批准 | 审核通过，展示中 |
| 已拒绝 | 审核拒绝，不展示 |

### 审核评论

- **批准**：点击「批准」按钮
- **拒绝**：点击「拒绝」按钮
- **删除**：点击「删除」按钮

### 评论设置

在「设置」->「评论设置」中配置：

```
☑ 开启评论
☑ 评论需要审核
☐ 允许游客评论
☑ 评论最大嵌套层数：3
```

### 邮件通知

评论通过审核后，系统会发送邮件通知博主。

## 订阅管理

### 订阅列表

1. 点击左侧菜单「订阅」
2. 查看所有订阅者

### 导出订阅者

点击「导出」按钮，下载 CSV 格式的订阅者列表。

### 发送邮件

1. 点击「发送邮件」
2. 填写主题和内容
3. 点击「发送」

### 邮箱配置

在「设置」->「邮件设置」中配置 SMTP：

```
邮件驱动：SMTP
发件服务器：smtp.example.com
端口：587
加密：TLS
用户名：your@email.com
密码：your-password
发件人名��：博客名称
发件人邮箱：noreply@example.com
```

## 用户管理

### 创建用户

1. 点击左侧菜单「用户」
2. 点击「新建用户」
3. 填写用户信息：

```
用户名：编辑
邮箱：editor@example.com
密码：********
角色：编辑（有权限管理文章）
```

### 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 完全控制 |
| 编辑 | 管理文章、评论、分类、标签 |
| 订阅者 | 订阅博客、评论 |

### 编辑用户

1. 点击用户名称
2. 修改信息
3. 点击「保存」

### 删除用户

1. 选择要删除的用户
2. 点击「删除」按钮
3. 确认删除（文章会转移给管理员）

## 前台使用指南

### 浏览文章

1. 打开博客首页
2. 浏览文章列表
3. 点击文章标题阅读详情

### 搜索文章

1. 在搜索框输入关键词
2. 按回车或点击搜索图标
3. 查看搜索结果

### 按分类浏览

1. 点击导航栏分类
2. 查看该分类下的所有文章

### 按标签浏览

1. 点击文章下的标签
2. 查看包含该标签的所有文章

### 评论互动

1. 阅读文章后，滚动到底部
2. 填写评论表单
3. 点击「提交」

### 订阅博客

1. 点击导航栏「订阅」
2. 输入邮箱
3. 点击「订阅」
4. 查收确认邮件，点击确认链接

### 取消订阅

1. 点击邮件底部的「取消订阅」链接
2. 点击确认

## 常见问题

### 无法登录

1. 检查邮箱和密码是否正确
2. 检查大小写
3. 尝试使用「忘记密码」功能重置

### 文章不显示

1. 检查文章状态是否为「已发布」
2. 检查发布时间是否已过
3. 检查文章分类是否正确

### 图片上传失败

1. 检查文件大小是否超过限制（默认 2MB）
2. 检查文件格式是否为图片格式
3. 检查 storage 目录是否可写

### 评论无法提交

1. 检查是否开启了评论功能
2. 检查是否需要审核
3. 尝试稍后再提交

### 邮件发送失败

1. 检查 SMTP 配置是否正确
2. 检查发件邮箱是否开启了 SMTP
3. 检查防火墙是否阻止了发件端口

### 性能问题

1. 启用缓存
2. 优化图片
3. 开启 OpCache
4. 使用 CDN
